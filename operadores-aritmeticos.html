Operadores Aritméticos

O que são Operadores Aritméticos?

Operadores aritméticos são símbolos usados para realizar operações matemáticas básicas em programação. Eles permitem que você execute cálculos com números e variáveis numéricas em seu código.

Principais Operadores Aritméticos

  • Adição (+): soma dois valores
  • Subtração (-): subtrai o segundo valor do primeiro
  • Multiplicação (*): multiplica dois valores
  • Divisão (/): divide o primeiro valor pelo segundo
  • Módulo (%): retorna o resto da divisão
  • Exponenciação (** ou ^): eleva um número a uma potência

Ordem de Precedência

A ordem de precedência dos operadores aritméticos é:

  1. Parênteses ()
  2. Exponenciação
  3. Multiplicação e Divisão (da esquerda para a direita)
  4. Adição e Subtração (da esquerda para a direita)

Importância dos Operadores Aritméticos

  • Permitem realizar cálculos matemáticos no código
  • São essenciais para processamento de dados numéricos
  • Facilitam a implementação de fórmulas e algoritmos
  • São utilizados em praticamente todos os tipos de programas

fn main() {
    // Usando operadores aritméticos para calcular a média de três números
    let num1 = 10;
    let num2 = 15;
    let num3 = 20;

    let soma = num1 + num2 + num3;
    let media = soma as f64 / 3.0;

    println!("A média de {}, {} e {} é: {}", num1, num2, num3, media);

    // Exemplo com outros operadores
    let resultado = (num1 * num2) - (num3.pow(2) % 7);
    println!("Resultado da expressão complexa: {}", resultado);
}
                    

public class OperadoresAritmeticos {
    public static void main(String[] args) {
        // Usando operadores aritméticos para calcular a média de três números
        int num1 = 10;
        int num2 = 15;
        int num3 = 20;

        int soma = num1 + num2 + num3;
        double media = (double) soma / 3;

        System.out.printf("A média de %d, %d e %d é: %.2f%n", num1, num2, num3, media);

        // Exemplo com outros operadores
        int resultado = (num1 * num2) - (int) Math.pow(num3, 2) % 7;
        System.out.println("Resultado da expressão complexa: " + resultado);
    }
}
                    

#include <iostream>
#include <cmath>

int main() {
    // Usando operadores aritméticos para calcular a média de três números
    int num1 = 10;
    int num2 = 15;
    int num3 = 20;

    int soma = num1 + num2 + num3;
    double media = static_cast<double>(soma) / 3;

    std::cout << "A média de " << num1 << ", " << num2 << " e " << num3 << " é: " << media << std::endl;

    // Exemplo com outros operadores
    int resultado = (num1 * num2) - static_cast<int>(std::pow(num3, 2)) % 7;
    std::cout << "Resultado da expressão complexa: " << resultado << std::endl;

    return 0;
}
